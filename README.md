# 🎲 Guess The Number Game

A simple Python game where the player tries to guess a random number generated by the computer.

## 📌 Description

- The computer randomly chooses a number between **1 and 10**.
- The player enters a number between **1 and 10**.
- If the player's number matches the computer's number, the player wins.
- Otherwise, the player loses.
- Entering `0` exits the game.

## 🚀 Features

- Random number generation using Python's `random` module.
- Input validation with `try-except`.
- Prevents invalid inputs (non-numeric values).
- Allows the player to play multiple rounds until they choose to exit.

## 🛠 Requirements

- Python 3.x
